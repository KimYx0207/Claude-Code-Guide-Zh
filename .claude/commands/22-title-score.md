# 📊 标题评分器

**功能**：输入标题，分析7个维度，给出爆款概率评分（0-100分）并提供改进建议

**评分维度**（数据验证版）：
1. 品牌词（35分）- 是否包含热门品牌 [1.59x]
2. 工具推荐公式（20分）- "用了X才知道+神器"模式 [5.25x最强！]
3. 动作词（15分）- 是否有行动指引 [1.95x]
4. 效率词（10分）- 是否承诺效率 [1.68x]
5. 问题解决（10分）- 是否解决痛点 [1.65x]
6. 数字/版本（10分）- 是否有具体数字
7. 标题长度（5分）- >25字效果更好（已验证）

**⚠️ 已删除**：情绪词（数据验证0.32x负相关）

---

## ⚡ 使用方法

```
/22-title-score [标题]
```

**示例**：
```

/22-title-score 这个AI工具太强了
/22-title-score Cursor 0.50 老金实测报告，亮点和避坑都在了
```

## 📊 评分标准

| 评级 | 分数 | 爆款概率 |
|------|------|----------|
| S | ≥85 | 🔥🔥🔥 极高(>80%) |
| A | 70-84 | 🔥🔥 较高(60-80%) |
| B | 55-69 | 🔥 中等(40-60%) |
| C | 40-54 | ⚠️ 较低(20-40%) |
| D | <40 | ❌ 很低(<20%) |
