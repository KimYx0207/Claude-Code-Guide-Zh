# ✅ 发文前检查清单

**功能**：发布前最后一道防线，8个维度全面检查文章是否满足爆款条件

**基于爆款的3个必要条件**：
1. 踩中真实痛点
2. 时效性（热点24小时内）
3. 标题含品牌词+动作词

---

## ⚡ 使用方法

```
/23-pre-check [文章文件路径]
```

**示例**：
```
/23-pre-check articles/2025-12-08-claude-update.md
```

---

## 📋 8个检查维度

| 维度 | 满分 | 说明 | 通过标准 |
|------|------|------|----------|
| 品牌词 | 25分 | Claude/Cursor/Gemini等 | 必须包含 |
| 动作词 | 15分 | 手把手/教你/一键等 | 必须包含 |
| 情绪词 | 10分 | 神器/绝了/真香等 | 建议包含 |
| 痛点匹配 | 15分 | 解决用户真实问题 | 有问句或痛点词 |
| 时效性 | 15分 | 最新/刚发布/版本号 | 建议包含 |
| 实用性 | 10分 | 代码/步骤/配图 | 至少2项 |
| 独特性 | 10分 | 与已有文章不重复 | 相似度<50% |
| 传播潜力 | 10分 | 长度适中+互动引导 | ≥5分 |

---

## ⚠️ 通过标准

**必须满足**（否则不建议发）：
- ✅ 品牌词检查通过
- ✅ 动作词检查通过
- ✅ 总分 ≥ 60分

**建议满足**：
- ✅ 痛点匹配
- ✅ 时效性
- ✅ 实用性
