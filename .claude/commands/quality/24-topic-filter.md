# 🎯 选题过滤器 V3.0 - 双轨制

**功能**：在动手写作前，判断选题分类和时效性，给出写作策略

**V3核心逻辑**：
1. **核心工具类**：大厂 + AI垂直厂商的工具 → 稳定流量，常写
2. **泛AI话题类**：AI现象/趋势/热点 → 破圈潜力，精选写

---

## ⚡ 使用方法

```
/24-topic-filter [选题描述]
```

**示例**：
```
/24-topic-filter Claude更新了新功能
/24-topic-filter Google Nano Banana做PPT
/24-topic-filter AI会取代程序员吗
```

---

## 🔧 执行流程

### 步骤1：产品时效性预检（重要！）

**如果选题涉及具体产品名称**，先用WebSearch搜索确认该产品是否是近期（一个月内）发布的：

```
搜索：[产品名] 发布时间 release date
```

判断逻辑：
- 如果产品是**一个月内**发布的 → 在context里加上"新发布"
- 如果产品是**一个月以上**发布的 → 不加额外context

**示例**：
- 选题"Google Nano Banana做PPT" → 搜索"Nano Banana Pro release date"
- 发现是2024年12月6日发布 → 一个月内 → context加"新发布"

### 步骤2：调用选题过滤器

```bash
cd ".claude/skills/gongzhonghao-writer/scripts" && python -X utf8 topic_filter.py "[选题]" "[context]"
```

**注意**：如果步骤1判断是新产品，context传"新发布"

### 步骤3：输出过滤结果

直接输出脚本的结果报告，格式如下：

```
============================================================
🎯 选题过滤器 V3.0 - 双轨制
============================================================

📝 选题：[选题内容]

------------------------------------------------------------
📂 分类：核心工具类 / 泛AI话题类 / 不建议
✅ 判断：值得写 / 不建议写
🔧 工具：[匹配到的工具]（如有）
📊 历史平均阅读：[数据]（如有）
⏰ 时效性：热点期 / 常青期

------------------------------------------------------------
📋 策略：[写作策略建议]
------------------------------------------------------------

💡 分析：
  [分析要点列表]

============================================================
```

## ⏰ 时效性判断

### 热点期（快写）
- 热点信号词：发布、上线、更新、推出、今天、昨晚、最新、官宣、重磅...
- 重大新闻词：估值、融资、收购、警报、慌了、炸了...
- 版本号：v5, 2.0, 3.1等
- **近期新产品**：一个月内发布的产品（需WebSearch确认）

### 常青期（慢写）
- 教程、指南、对比、盘点
- 非时效性内容
