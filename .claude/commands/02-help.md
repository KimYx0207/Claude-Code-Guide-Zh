## 🚀 推荐工作流

### 流程1：日常写作（推荐新手）
```
/topic-filter [选题]     # 1. 先过滤选题可行性
       ↓
/write [主题]            # 2. 完整写作流程
       ↓
/pre-check               # 3. 发文前最终检查
       ↓
/image 或 /infographic   # 4. 可选：添加配图
```

### 流程2：快速产出（追热点）
```
/hotspot                 # 1. 扫描今日热点
       ↓
/write-auto [热点]       # 2. 全自动爆款生成
```

### 流程3：数据驱动（长期优化）
```
/data-collect            # 1. 收集历史数据
       ↓
/data-analyze            # 2. 分析爆款规律
       ↓
/title-gen [主题]        # 3. 根据规律生成标题
```

### 流程4：标题优化（提升点击率）
```
/title-gen [主题]        # 1. 生成5个标题候选
       ↓
/title-score [标题]      # 2. 逐个评分，选最高分
```

## 📊 核心规则速览

### 爆款标题3必要条件
1. **必须有品牌词**：Claude/Cursor/Windsurf/GPT等
2. **必须有动作词**：放弃/换/直接用/发现/试了下等
3. **标题评分≥60分**：7维度综合评分

### 7维度标题评分
| 维度 | 分值 | 示例 |
|------|------|------|
| 品牌词 | 35分 | Claude, Cursor, Windsurf |
| 动作词 | 15分 | 放弃, 换, 直接用 |
| 效率词 | 10分 | 秒, 直接, 一键 |
| 情绪词 | 15分 | 惊, 服, 震惊, 牛逼 |
| 问题解决 | 10分 | 搞定, 解决, 轻松 |
| 数字/版本 | 10分 | 3.5, 90%, 10倍 |
| 长度 | 5分 | 15-25字最佳 |

### 9维度质量检测
| 维度 | 阈值 | 说明 |
|------|------|------|
| AI腔检测 | <20分 | 越低越好 |
| 自然度 | >80分 | 越高越好 |
| 真诚度 | >75分 | 越高越好 |
| 啰嗦度 | <25分 | 越低越好 |
| 重复度 | <15% | 越低越好 |
| 可读性 | >85分 | 越高越好 |
| 人味儿指数 | >70分 | 接地气 |
| 情感真实性 | >75分 | 真实情感 |
| 脏话检测 | =0处 | 零容忍 |

## 💡 常见问题

**Q: 新手该用哪个命令？**
A: 先用 `/topic-filter` 验证选题，再用 `/write` 完整流程

**Q: 怎么快速追热点？**
A: 用 `/hotspot` 扫描热点，然后 `/write-auto` 一键生成

**Q: 标题不够爆怎么办？**
A: 用 `/title-gen` 生成候选，用 `/title-score` 逐个评分优化

**Q: 发布前要检查什么？**
A: 必须运行 `/pre-check`，确保品牌词+动作词+评分≥60
