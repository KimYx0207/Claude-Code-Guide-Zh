## 一、选题类型及预期（必须记住！）

| 选题类型 | 预期阅读 | 转发率 | 优先级 | 建议频率 |
|---------|---------|--------|--------|---------|
| **新品深度测评** | 5000-15000 | 中 | ⭐⭐⭐⭐⭐ | 有热点就写 |
| **降价/优惠信息** | 4000-8000 | 高(300+) | ⭐⭐⭐⭐⭐ | 有就写 |
| **个人经验故事** | 4000-7000 | 中高 | ⭐⭐⭐⭐ | 每月2-3篇 |
| **实用技巧合集** | 3000-5000 | 中 | ⭐⭐⭐ | 每月1-2篇 |
| 功能介绍 | 2000-3000 | 低 | ⭐⭐ | 尽量少写 |
| 教程类 | 2000-3000 | 低 | ⭐⭐ | 工具文定位 |
| 公司/行业分析 | 1500-2500 | 低 | ⭐ | 除非有独家 |

**结论**：主攻测评、降价、经验故事，少写功能介绍和教程！

## 二、每周选题配比（黄金比例）

### 推荐配比（每周3篇）

```
必发（2篇）：
├── 测评类 x 1（有热点时）
└── 降价/经验故事 x 1

可选（1篇）：
├── 技巧合集
├── 深度分析（有独家角度时）
└── 或留空（宁缺毋滥）
```

### 选题日历模板

| 周一 | 周三 | 周五 |
|------|------|------|
| 测评类（追热点） | 经验故事/技巧 | 降价/优惠 |

**注意**：
- 发文间隔至少48小时
- 同类型文章间隔至少1周
- 有大热点时打破常规，优先发热点

## 三、选题评估标准（5分制）

每个选题发布前，用这个标准评估：

| 评估维度 | 5分 | 3分 | 1分 |
|---------|-----|-----|-----|
| **时效性** | 12小时内 | 24-48小时 | 超过48小时 |
| **受众面** | 所有AI用户 | 特定工具用户 | 极小众 |
| **实用性** | 立即可用 | 有参考价值 | 纯知识 |
| **独特性** | 独家角度 | 有个人观点 | 和别人一样 |
| **情绪点** | 强共鸣/争议 | 有兴趣 | 平淡 |

**评分标准**：
- 20-25分：必发！抢时间
- 15-19分：可发，优化后发
- 10-14分：考虑是否有更好选题
- <10分：不发！

## 四、选题来源及方法

### 来源1：热点驱动（最高优先级）

```
触发条件：
- 大模型发布新版本
- 工具重大更新
- 价格变动/优惠活动
- 技术突破/论文

操作：
1、发现热点（见热点监控清单）
2、评估是否值得写（用5分制评估）
3、如果≥15分，立即启动写作
```

### 来源2：个人经验积累

```
触发条件：
- 使用某工具有了新发现
- 踩了一个坑并解决了
- 积累了可总结的经验

操作：
1、记录下来（用备忘录/Notion）
2、积累3-5个点后写成文章
3、适合经验故事类/技巧合集类
```

### 来源3：用户痛点挖掘

```
触发条件：
- 看到用户抱怨/吐槽
- 发现常见问题/误区
- 有人问你问题

来源渠道：
- 评论区
- 私信
- Reddit/知乎问题
- 朋友圈讨论

操作：
1、收集用户真实痛点
2、写成解决方案类文章
3、适合技巧类/避坑类
```

### 来源4：竞品对比机会

```
触发条件：
- 两个产品打架
- 新品挑战老牌
- 功能高度重合

操作：
1、做真实对比测试
2、写成对比评测文章
3、有争议点更好
```

## 五、选题红线（绝对不碰）

### ❌ 绝对不写

1. **时效性超过1周的新品介绍**
   - 别人都写完了，你再写没价值

2. **纯翻译/搬运的内容**
   - 没有个人观点和体验的二手信息

3. **太小众的工具（用户<1万）**
   - 受众面太窄，天花板低

4. **纯负面吐槽（没有解决方案）**
   - 读者看完没有收获

5. **蹭无关热点**
   - 强行关联只会掉粉

### ⚠️ 谨慎写

1. **公司/行业分析**
   - 除非有独家信息，否则阅读预期低

2. **长篇教程**
   - 适合作为工具文沉淀，不追流量

3. **争议性话题**
   - 要有真实观点，不要骑墙

## 七、选题库管理

### 建立选题库

用Notion/飞书文档建立选题库，包含：

| 字段 | 说明 |
|------|------|
| 选题名称 | 简要描述 |
| 类型 | 测评/降价/经验/技巧 |
| 来源 | 热点/经验/用户痛点 |
| 时效性 | 有时效/无时效 |
| 评估分数 | 5分制总分 |
| 状态 | 待写/写作中/已发布 |
| 备注 | 相关素材链接 |

### 选题库维护

```
每日：
- 添加新发现的选题
- 更新时效性状态

每周：
- 清理过期选题
- 评估本周发文效果
- 调整选题策略
```

## 八、本周选题自检

每周一问自己：

```
□ 本周有没有追到热点？
□ 有没有发降价/优惠类？
□ 有没有发经验故事类？
□ 是不是又写了功能介绍/教程？
□ 选题评估分数是否≥15分？
```

---

**记住**：选题决定了文章的天花板。测评/降价/经验故事天花板高，功能介绍/教程天花板低。把时间花在高天花板的选题上！
