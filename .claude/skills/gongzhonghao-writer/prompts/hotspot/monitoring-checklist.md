```

"Google" AND ("Gemini" OR "AI" OR "release")
```

## 五、热点响应SOP

### 发现热点后的操作流程

```
0-30分钟：确认热点
├── 核实信息源（官方为准）
├── 评估热度（HN排名/Twitter转发）
└── 判断是否值得写（见选题策略）

30-60分钟：快速Research
├── 官方博客/更新日志
├── Reddit/HN用户反馈
├── Twitter讨论
└── 竞品对比信息

1-3小时：写作
├── 使用对应文章模板
├── 边写边补充Research
└── 完成初稿

3-4小时：检查发布
├── 真实性检查
├── 标题优化
├── 质量检测
└── 发布
```

### 热点优先级判断

| 热点类型 | 优先级 | 响应时间 |
|---------|--------|---------|
| 大模型发布（GPT-5/Claude 4级别） | P0 | 6小时内 |
| 重要功能更新（API/价格变化） | P0 | 12小时内 |
| 工具重大更新（Cursor V2级别） | P1 | 24小时内 |
| 研究论文/技术突破 | P2 | 48小时内 |
| 公司动态/融资新闻 | P3 | 可不追 |
