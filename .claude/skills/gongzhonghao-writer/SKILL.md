# 公众号写作助手 Skill

**版本**：V9.0 | **更新**：2025-12-20
**类别**：内容创作
**触发关键词**：写作、公众号、文章生成、热点、标题、爆款

---

## 功能概述

基于82篇文章数据验证的三层架构爆款系统，配置驱动、优先级公式、Layer1/2/3选题过滤。

---

## 三层架构（核心）

| Layer | 定义 | 平均阅读 | 爆款率 |
|-------|------|---------|--------|
| Layer1 | 核心工具官方 | 2,200 | 71% |
| Layer2 | 核心工具生态 | 1,450 | 24% |
| Layer3 | 泛AI话题 | 908 | 5% |

**优先级公式**：
```
priority = layer_score × timeliness × type_weight × brand_tier ÷ risk
```

---

## 核心能力

### 1. 标题生成（5大公式）
- 工具推荐型（有效性5.25x）
- 教程词型（有效性1.95x）
- 数据发现型（有效性1.78x）
- 品牌动作型（有效性1.65x）
- 问题解决型（有效性1.52x）

### 2. 质量检测（9维度）
- AI腔检测(<20分) | 自然度(>80分) | 真诚度(>75分)
- 啰嗦度(<25分) | 重复度(<15%) | 可读性(>85分)
- 人味儿指数(>70分) | 情感真实性(>75分) | 脏话检测(=0)

### 3. 选题过滤（三层架构）
- `/topic-filter` - 优先级分数计算
- `/scan` - 三层架构热点扫描

### 4. 热点扫描
- GitHub生态项目自动过滤
- 只推荐Layer1/Layer2项目

## 快速使用

```bash
/help              # 查看所有命令
/topic-filter [选题]  # 三层架构选题过滤
/write [主题]      # 完整写作流程
/scan              # 热点扫描
```
