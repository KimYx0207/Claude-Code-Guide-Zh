# 降价类文章模板（爆款公式2）

**预期阅读**：4000-8000
**预期转发**：300-450+（高转发利器！）
**核心要求**：优惠信息要准确、及时、可操作

---

## 标题公式（4选1）

```
1. 降价冲击型：产品名 + 降价数字 + 情绪词 + 行动
   示例：年付降价389，整个人都麻了
   示例：Cursor降价200块，赶紧上车

2. 限时紧迫型：产品名 + 限时 + 优惠力度 + 情绪词
   示例：Claude Pro限时5折，错过等一年
   示例：即梦会员首月1块，今天最后一天

3. 对比冲击型：原价 vs 现价 + 产品名 + 情绪词
   示例：从199降到99，这个AI工具疯了
   示例：省了500块，Claude年付真香

4. 福利分享型：发现一个羊毛 + 产品名 + 具体优惠
   示例：薅到一个大羊毛，Cursor首年便宜200
   示例：这个优惠码太香了，Claude Pro打7折
```

**标题检查**：
- [ ] 有具体降价数字？
- [ ] 有情绪词（麻了/香了/疯了）？
- [ ] 有行动号召暗示？
- [ ] 长度15-25字？

---

## 文章结构（3段式，短平快）

### 第1段：优惠信息直给（3-5行）

**必须包含**：
- ✅ 产品名称
- ✅ 原价 vs 现价
- ✅ 优惠力度（百分比/具体金额）
- ✅ 有效期限（如果有）

**模板**：
```
[产品名][活动类型]了！

原价[X]，现在只要[Y]，直接省[Z]块。

[有效期说明，如果有限时]
```

**示例**：
```
Cursor年付降价了！

原价192刀，现在只要144刀，直接省了48刀（约350块）。

活动到月底，之后恢复原价。
```

❌ **禁止**：
- 铺垫太长才说优惠
- 优惠信息不清晰
- 没有具体数字

---

### 第2段：价值分析（10-15行）

**必须包含**：
- ✅ 这个产品值不值这个价
- ✅ 和竞品对比（价格/功能）
- ✅ 适合什么人买
- ✅ 不适合什么人买（真诚建议）

**模板**：
```
**值不值得买？**

先说结论：[值/看情况/不值]

[产品名]的核心价值是[1-2个核心功能]，
对比[竞品1]的[价格]、[竞品2]的[价格]，
这个价格[评价]。

**适合买的人**：
- [人群1]
- [人群2]

**不建议买的人**：
- [人群1]
- [人群2]
```

**示例**：
```
**值不值得买？**

先说结论：重度用户值，轻度用户等等。

Cursor的核心价值是AI代码补全+多文件编辑，
对比GitHub Copilot的100刀/年、Windsurf的免费，
144刀/年这个价格算中等偏上。

**适合买的人**：
- 每天写代码超过4小时的
- 需要AI辅助重构大项目的
- 试用期用爽了的

**不建议买的人**：
- 偶尔写写代码的（免费版够用）
- 主要用Python的（Copilot可能更好）
- 还没试用过的（先试用再说）
```

---

### 第3段：购买指南 + 行动号召（5-10行）

**必须包含**：
- ✅ 具体购买步骤（如果有坑要说）
- ✅ 优惠码/链接（如果有）
- ✅ 注意事项（续费价格、取消政策等）
- ✅ 行动号召

**模板**：
```
**怎么买？**

1、[步骤1]
2、[步骤2]
3、[步骤3]

**注意**：[注意事项，如续费价格、自动续费等]

[行动号召，如"需要的赶紧上车"]
```

**示例**：
```
**怎么买？**

1、打开Cursor官网（cursor.com）
2、点击Pricing，选Pro年付
3、用国内信用卡就能付（Visa/Mastercard）

**注意**：
- 默认开了自动续费，不想续的记得关
- 首年优惠，续费恢复原价192刀
- 学生有额外折扣，别忘了申请

现在买最划算，犹豫的话活动结束就没了。
```

## 额外加分项（可选）

### 加分项1：个人使用体验（3-5行）
```
**我自己用了[时间]的体验**：

[1-2个真实使用场景]
[省了多少时间/提升了多少效率]
```

### 加分项2：社区反馈（3-5行）
```
**其他用户怎么说**：

Reddit网友@xxx："..."
Twitter用户@xxx："..."
```

### 加分项3：替代方案（3-5行）
```
**如果觉得贵，这些替代方案也行**：

- [替代品1]：[价格]，[优缺点]
- [替代品2]：[价格]，[优缺点]
```
