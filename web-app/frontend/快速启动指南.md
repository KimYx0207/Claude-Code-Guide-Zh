**版本**: V1.0 | **日期**: 2025-12-14

## 🚀 一键启动

### 方法1: 开发模式（推荐）
```bash
cd web-app/frontend
npm run dev
```
访问: http://localhost:3000

### 方法2: 生产模式
```bash
cd web-app/frontend
npm run build
npm start
```

### 测试顺序

## 🔧 常见问题

### Q1: 端口3000被占用？
```bash
# 方法1: 杀死占用进程
lsof -ti:3000 | xargs kill

# 方法2: 使用其他端口
npm run dev -- -p 3001
```

### Q2: 编译失败？
```bash
# 删除缓存重新构建
rm -rf .next
npm run build
```

### Q3: API返回500错误？
```bash
# 检查数据文件路径
ls -la ../../data/
ls -la ../../docs/课程资料包/
```

### Q4: 依赖安装失败？
```bash
# 清理缓存重新安装
rm -rf node_modules package-lock.json
npm install
```

## 🎓 进阶使用

### 自定义配置
编辑 `next.config.js`:
```js
module.exports = {
  env: {
    API_BASE_URL: 'http://localhost:8000'
  }
}
```

### 添加新API
在 `app/api/` 下创建新路由:
```typescript
// app/api/custom/route.ts
export async function GET() {
  return NextResponse.json({ success: true });
}
```

### 扩展Tab
在 `page.tsx` 中添加新Tab组件:
```typescript
function NewTab() {
  return <div>新功能</div>;
}
```
