Claude Code 是一个强大的 AI 编程助手，可以帮助您更高效地编写代码、调试问题和学习新技术。

### 🚀 快速开始

#### 启动 Claude Code

在项目目录中打开终端，运行：

```bash
claude
```

这将启动交互式对话界面。

### 💬 基本对话模式

#### 1. 提问代码问题

直接向 Claude Code 提问：

```bash
你: 如何在 JavaScript 中实现防抖函数？

Claude: 我可以帮你实现一个防抖函数...
```

#### 2. 代码生成

让 Claude Code 生成代码：

```bash
你: 创建一个 React 组件，显示用户列表

Claude: 我会为你创建一个 UserList 组件...
```

#### 3. 代码解释

请求解释现有代码：

```bash
你: 解释一下这段代码的作用
[粘贴代码]

Claude: 这段代码的主要功能是...
```

### 🛠️ 常用命令

#### 文件操作

```bash
# 读取文件
claude "读取 src/index.js 文件"

# 创建文件
claude "创建一个新的 utils.js 文件，包含常用工具函数"

# 修改文件
claude "修改 package.json，添加新的依赖项"
```

#### 代码重构

```bash
# 重构代码
claude "重构 src/components/Header.js，使用 hooks 代替 class"

# 优化性能
claude "优化这个组件的性能"
```

#### 调试帮助

```bash
# 查找错误
claude "为什么这段代码报错？"

# 修复 bug
claude "修复这个 TypeError"
```

### 📝 最佳实践

#### 1. 清晰的指令

❌ 不好的例子：

```bash
做个网站
```

✅ 好的例子：

```bash
创建一个响应式导航栏组件，包含 logo、菜单项和移动端汉堡菜单
```

#### 2. 提供上下文

给 Claude Code 提供足够的上下文信息：

```bash
我正在开发一个电商网站，使用 React 和 TypeScript。
请帮我创建一个购物车组件，需要显示商品列表、数量控制和总价计算。
```

#### 3. 迭代优化

不要期望一次性得到完美结果，可以逐步优化：

```bash
你: 创建一个登录表单
Claude: [生成基础表单]

你: 添加表单验证
Claude: [添加验证逻辑]

你: 添加记住我功能
Claude: [添加记住我选项]
```

### ⚡ 效率技巧

#### 1. 使用简短别名

在 shell 配置文件中添加：

```bash
alias cc="claude"
```

#### 2. 批量处理

可以在一次对话中处理多个任务：

```bash
请帮我完成以下任务：
1. 创建 User 模型
2. 实现 CRUD API
3. 添加错误处理
4. 编写 API 文档
```

#### 3. 保存常用 Prompts

将常用的提示词保存为文件，需要时快速调用。
