### 核心功能介绍

#### 工作原理

「Happy」的核心工作机制非常简单：

1. 远程连接：手机远程连接自己电脑的终端
2. 指令传输：通过手机向远程电脑发送指令
3. 本地执行：电脑上的 Claude Code 执行任务
4. 结果返回：执行完成后将结果返回到手机

#### 三大核心优势

1. 安装便捷：一键安装，无需复杂配置
2. 深度集成：专为 Claude Code 打造的完整体验
3. 完全免费：开源项目，支持自部署服务器

#### 实用场景

##### 1. 随时了解开发进度

当您需要离开电脑时，可以：

```bash
> 请帮我完成用户认证功能的开发，完成后通知我
```

然后通过手机实时查看开发进度，即使在下楼抽烟或出门办事时也能掌握项目状态。

进阶技巧：集成第三方通知服务（如飞书机器人），实现完成自动通知：

```bash
> 开发完成后，请更新飞书文档状态并发送通知
```

##### 2. 从想法到上线全流程

出门在外时突然有灵感？没问题！

```bash
> 帮我开发一个在线留言板功能，开发完成后通过 ngrok 生成访问地址
```

1. Claude Code 自动开发功能
2. 通过 ngrok 生成在线访问地址
3. 手机直接测试验证
4. 通过 Git 提交代码：git commit && git push
5. 自动部署到生产环境（集成 Vercel 等）

##### 3. 文档生成与管理

对于本地自动化流程生成的文档（图片/PDF等），可以通过以下方式管理：

```bash
> 生成今日项目报告并上传到 GitHub，返回访问链接
```

这样就可以在手机上直接访问、下载和管理生成的文档。

### 保姆级安装指南

#### 第一步：安装 Happy

在终端中执行以下命令：

```bash
npm i -g happy-coder && happy
```

#### 第二步：手机连接

1. 选择连接类型：在电脑上选择 “Mobile App” 类型
2. 扫描二维码：手机扫描电脑显示的二维码
3. 下载应用：在应用商店搜索 “Happy：Claude Code Client”

> iOS 用户：国内 App Store 可直接下载，无需海外账号

iOS 用户：国内 App Store 可直接下载，无需海外账号

#### 第三步：开始使用

连接成功后，手机界面会显示与电脑相同的对话记录，可以直接在手机上发送指令。

### Happy 使用方法详解

#### 基础操作

##### 对话与执行过程查看

- 发送指令：支持文字和语音输入
- 查看过程：点击可查看网络搜索、子代理、MCP等执行详情
- 界面清爽：隐藏冗余信息，需要时才查看详情

##### 斜杠命令支持

直接输入 / 符号即可调出斜杠命令菜单：

```bash
/help          # 查看帮助
/agents        # 管理子代理
/status        # 查看状态
/init          # 初始化项目
```

##### 任务中断

点击【感叹号】按钮中断任务执行，相当于电脑上的 ESC 键。

#### 高级功能

##### 权限与模型设置

点击【齿轮】按钮快速切换：

- 操作权限：文件读写、网络访问等权限设置
- 模型切换：在不同 AI 模型间切换
- Yolo 模式：一键开启全自动模式（相当于 --dangerously-skip-permissions）

##### 设置重置

如需完全重置 Happy：

1. 删除用户目录下的 .happy 文件夹
2. 执行 happy 命令重启客户端

### 移动端编程最佳实践

#### 1. 任务规划

在手机上更适合执行以下类型的任务：

- 代码审查：请审查最近的代码提交
- 文档生成：为这个功能生成使用文档
- 测试执行：运行测试套件并报告结果
- 简单修复：修复登录页面的样式问题

#### 2. 环境配置

为确保最佳体验，建议：

```bash

git config --global user.name "你的名字"
git config --global user.email "你的邮箱"

# 安装必要工具
npm install -g ngrok  # 用于内网穿透
```

#### 3. 安全考虑

- 网络安全：确保在安全的网络环境下使用
- 权限控制：合理设置文件访问权限
- 数据备份：定期备份重要代码和配置
