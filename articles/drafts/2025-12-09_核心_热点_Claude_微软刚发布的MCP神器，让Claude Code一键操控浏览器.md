## Playwright MCP能做什么？

简单说，就是让AI控制浏览器。

它能做的事：

**浏览器操作**
打开网页、点击按钮、输入文字、滚动页面、截图。

**页面检查**
检查元素是否存在、读取文本内容、验证页面标题、检查网络请求。

**表单测试**
填写表单、上传文件、选择下拉框、勾选复选框。

**多页面操作**
打开新标签页、切换窗口、处理弹窗、关闭页面。

基本上人能在浏览器里做的事，它都能自动化。

## 怎么用？

两步搞定。

**第一步：安装Playwright**

```bash
npm install -D @playwright/test
npx playwright install
```

**第二步：配置MCP**

在`.mcp.json`里添加：

```json
{
  "mcpServers": {
    "playwright": {
      "command": "npx",
      "args": ["-y", "@playwright/mcp"]
    }
  }
}
```

重启Claude Code，试试：

"帮我打开baidu.com并截图"

如果浏览器自动打开并截图，就说明配置成功了。

## 当然也有短板

用了一周，发现几个问题。

**问题1：不能绕过所有反爬虫**

有些网站会检测自动化浏览器。

虽然Playwright比Selenium隐蔽很多，但还是会被检测到。

这种情况只能手动操作。

**问题2：复杂交互需要描述清楚**

如果网页交互很复杂，你要把每一步说清楚。

否则AI可能理解错误，点错地方。

**问题3：首次运行要下载浏览器**

Playwright会下载Chromium、Firefox、WebKit。

第一次运行时间会比较长。
