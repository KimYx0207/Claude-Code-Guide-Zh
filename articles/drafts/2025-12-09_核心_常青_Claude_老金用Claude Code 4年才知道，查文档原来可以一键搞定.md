## Context7是什么？

简单说，就是一个MCP服务器。

专门用来查询技术文档的。

它的特点：

**第一，文档总是最新的**。

你问的问题，它会去查最新版本的官方文档。

不像Claude自己的知识，可能是几个月前的。

**第二，覆盖面广**。

支持的库和框架太多了：React、Next.js、Vue、FastAPI、Django、TailwindCSS...

基本上你能想到的主流技术，它都有。

**第三，查询速度快**。

不是简单的搜索。

它会理解你的问题，找到最相关的文档部分。

然后把重点内容提取出来。

## 怎么用？

三步搞定。

**第一步：安装**

```bash
npx -y @upstash/context7-mcp
```

**第二步：配置**

在`.mcp.json`里添加：

```json
{
  "mcpServers": {
    "context7": {
      "command": "npx",
      "args": ["-y", "@upstash/context7-mcp"]
    }
  }
}
```

**第三步：使用**

重启Claude Code，试试问：

"React的useState怎么用？"

如果Claude能回答，并且引用了官方文档，就说明配置成功了。

## 当然也有短板

用了一周，发现几个问题。

**问题1：不是所有库都支持**

虽然覆盖面很广，但还是有些小众库查不到。

这时候只能回到传统方式，手动查文档。

**问题2：有时候查询结果太长**

如果问的问题太笼统，它会返回一大段文档。

需要你自己筛选重点。

所以问题要尽量具体。

**问题3：需要网络连接**

这个工具要实时查询文档。

如果网络不稳定，会查询失败。
