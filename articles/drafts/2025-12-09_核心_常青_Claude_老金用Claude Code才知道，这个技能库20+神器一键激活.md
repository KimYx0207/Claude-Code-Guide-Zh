## 怎么用？

三步搞定。

**第一步：克隆仓库**

```bash
git clone https://github.com/obra/superpowers.git
```

**第二步：安装**

```bash
cd superpowers
npm install
```

**第三步：配置Claude Code**

在`~/.config/claude/skills/`目录下创建链接：

```bash
ln -s /path/to/superpowers ~/.config/claude/skills/
```

重启Claude Code就行了。

不用额外配置，技能会自动激活。

## 当然也有短板

用了一周，发现几个问题。

**问题1：首次加载慢**

因为包含20+个技能，第一次加载时间会长一点。

大概需要10-15秒。

但之后就快了，会缓存起来。

**问题2：有些技能有依赖**

比如PDF处理需要安装额外的库。

Excel操作需要Node.js环境。

不过安装也简单，按提示装就行。

**问题3：技能可能冲突**

如果你已经装了其他Skills，可能会和superpowers冲突。

建议先卸载其他重复的Skills，只保留superpowers。
