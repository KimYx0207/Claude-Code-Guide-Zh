# 公众号写作助手 - 开课教程整合PRD

## 项目背景

用户即将开课，需要将现有的公众号写作助手项目资料整合成系统化的教学课程。

### 现有资料盘点

**GAC文章（43篇）**：Claude Code使用技巧文章（已有Markdown版本）

**Slash命令（20个）**：
- 帮助类：/help
- 写作类：/write, /write-auto, /write-rewrite
- 热点类：/hotspot, /daily
- 标题类：/title-gen, /title-score, /pre-check, /topic-filter
- 图片类：/image, /infographic
- 数据类：/data-collect, /data-analyze
- 工具类：/test-mcp, /ai-orchestrator
- Task Master快捷命令：4个

**提示词规范（12个）**：
- 核心：老金风格V6、爆款规律V7.2、选题策略V3.0
- 热点：热点源、热点监控、热点可信度、热点访问方法
- 配图：配图生成器、信息图生成器
- 其他：格式规范、写作检查清单

**Python脚本（20个）**：数据收集、标题生成/评分、质量检测、信息图生成等

## 项目目标

将分散的教程资料整合成系统化的开课教程，让学员能够：
1. 快速上手公众号写作助手
2. 掌握AI辅助写作的核心技巧
3. 学会数据驱动的写作优化
4. 能够独立完成爆款文章创作

## 课程结构设计（6大模块）

### 模块1：快速入门篇（2课时）

**目标**：让学员能够独立完成第一篇文章

**内容**：
- 第1课：环境配置与工具介绍（Claude Code + MCP）
  - 基于GAC文章：安装、配置、基础使用
  - Slash命令：/help, /test-mcp

- 第2课：写作第一篇文章（Hello World）
  - 核心命令：/write
  - 核心规范：老金风格V6（简化版）
  - 实战：从选题到发布完整流程

**输出物**：
- 《快速入门手册.md》
- 《第一篇文章实战指南.md》
- 配置检查清单

### 模块2：写作流程篇（3课时）

**目标**：掌握完整的写作工作流

**内容**：
- 第3课：选题策略与过滤
  - 命令：/topic-filter, /hotspot
  - 规范：选题策略V3.0（双轨制）
  - 实战：分析10个选题案例

- 第4课：标题创作与优化
  - 命令：/title-gen, /title-score
  - 规范：爆款规律V7.2（标题公式部分）
  - 工具：title_generator.py, title_scorer.py
  - 实战：生成并优化5个标题

- 第5课：内容创作与质量检测
  - 命令：/write, /pre-check
  - 规范：老金风格V6（完整版）、爆款规律V7.2
  - 工具：quality_detector.py, pre_publish_checker.py
  - 实战：完成一篇完整文章

**输出物**：
- 《写作流程完整指南.md》
- 《选题策略实战手册.md》
- 《标题优化工具包.md》
- 《质量检测标准.md》

### 模块3：进阶技巧篇（3课时）

**目标**：掌握提升效率和质量的进阶技巧

**内容**：
- 第6课：快速产出技巧
  - 命令：/write-auto, /write-rewrite, /daily
  - 场景：追热点、文章翻新、日更模式
  - 实战：24小时内完成热点文章

- 第7课：配图与可视化
  - 命令：/image, /infographic
  - 规范：配图规则、信息图规则
  - 工具：infographic_generator.py
  - 实战：为文章添加配图和信息图

- 第8课：多AI协作
  - 命令：/ai-orchestrator
  - 场景：复杂任务拆解、多引擎协作
  - 实战：使用Codex+Gemini完成复杂写作任务

**输出物**：
- 《进阶技巧速查表.md》
- 《配图与可视化指南.md》
- 《多AI协作实战.md》

### 模块4：数据驱动篇（2课时）

**目标**：学会用数据优化写作策略

**内容**：
- 第9课：数据收集与分析
  - 命令：/data-collect, /data-analyze
  - 工具：collect_wechat_data.py, analyze_wechat_data.py
  - 实战：收集和分析自己的历史数据

- 第10课：规律挖掘与应用
  - 规范：爆款规律V7.2（数据验证部分）
  - 工具：rule_validator.py, check_data_sync.py
  - 实战：根据数据调整写作策略

**输出物**：
- 《数据驱动工作流.md》
- 《数据分析报告模板》
- 《个人数据看板.md》

### 模块5：Claude Code深度篇（2课时）

**目标**：深度掌握Claude Code的高级功能

**内容**：
- 第11课：Hooks与Skills定制
  - 基于GAC文章：hooks-reference, skills-best-practices
  - 实战：创建自定义Hook和Skill

- 第12课：MCP服务器集成
  - 基于GAC文章：mcp-integration, plugin-marketplace-guide
  - 实战：集成第三方MCP服务

**输出物**：
- 《Claude Code高级功能手册.md》
- 《自定义Hooks与Skills指南.md》
- 《MCP集成实战.md》

### 模块6：综合实战篇（2课时）

**目标**：独立完成完整的写作项目

**内容**：
- 第13课：从0到1完整案例
  - 综合应用所有命令和技巧
  - 完整流程：选题 → 写作 → 优化 → 发布
  - 实战：独立完成一篇爆款文章

- 第14课：课程总结与进阶路径
  - 常见问题解答
  - 最佳实践总结
  - 进阶学习路径

**输出物**：
- 《完整案例演示.md》
- 《常见问题FAQ.md》
- 《进阶学习路径.md》
