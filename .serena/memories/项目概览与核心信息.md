# 微信公众号写作助手项目 - 核心信息

## 项目定位
专业的微信公众号内容创作系统，专注于AI编程推广和"老金风格"科技博主写作。

## 核心功能
1. **AI编程推广写作** - 降低编程门槛，让更多人体验AI编程乐趣
2. **老金风格创作** - 口语化表达，短句+空行，**粗体**强调，真实数据案例
3. **自然引导V4系统** - 从规则约束转向自然表达，去除AI腔，增加人情味
4. **SEO优化** - 微信搜一搜排名优化，关键词布局
5. **完整工作流** - 从选题调研到文章生成的7阶段自动化流程

## 技术架构
- **基础平台**: Claude Code + Skills系统
- **MCP服务**: Sequential Thinking, Context7, Exa Web Search, Memory等
- **写作引擎**: V4自然引导系统，8维度质量检测
- **自动化**: Windows批处理Hook，自动备份，风格验证
- **文件组织**: 自动文件管理和归档

## 老金风格核心要求
- **语言特点**: 每句15-20字内，一两句就换行，段落间加空行
- **表达风格**: 用"老金我"、"咱们"、"家人们"，像微信群聊天
- **内容要求**: 7步结构完整（反常识钩子→痛点→冲突→转折→硬结果→趋势→哲理）
- **固定签名**: "关注老金，AI 工具不迷路，咱们下回再接着扒！"
- **禁用词**: 赋能、降本增效、颠覆式、划时代、闭环、打法、生态、矩阵、链路

## 项目结构
- `.claude/skills/gongzhonghao-writer/` - V4写作技能核心
- `.claude/commands/` - 斜杠命令（/title, /outline, /expand等）
- `.claude/hooks/` - 自动化脚本（风格验证、备份等）
- `articles/` - 已发布的文章
- `gac_articles_markdown/` - GAC提示词技巧文档
- `.spec-workflow/` - 规范工作流系统

## 质量标准
- AI腔检测 < 20分
- 自然度 > 80分
- 真诚度 > 75分
- 啰嗦度 < 25分
- 重复度 < 15%
- 可读性 > 85分
- 人味儿 > 70分

## 使用场景
- 公众号文章创作（AI工具测评、技术趋势、实战教程）
- 自媒体内容生成
- 零门槛编程指南
- 老金风格科技博主写作
