# Articles 目录分类说明

**版本**：V2.0（修正版）
**更新日期**：2025-12-12

---

## 正确的目录结构

```
articles/
├── drafts/       # 未发布草稿（当前所有文章）
│   └── *.md
└── README.md     # 本文件

archive/          # 已发布文章（项目根目录）
├── 2024/
└── 2025/

docs/课程资料包/  # 教程文档（项目根目录）
├── 模块1-8/
├── 模块9/
└── 附录A/B/C/D/
```

---

## 分类说明

### articles/drafts/ - 未发布草稿

**当前内容**：35篇待发布文章

**文件名格式**：
```
YYYY-MM-DD_[分类]_[时效]_[品牌]_标题.md
```

**使用流程**：
1. `/write [主题]` → 自动保存到drafts/
2. `/pre-check` → 质量检查
3. 发布到公众号 → 手动移至archive/
4. 定期清理超过60天的草稿

---

### archive/ - 已发布文章（根目录）

**存放位置**：项目根目录`archive/`

**内容**：已在微信公众号发布的文章

**管理规则**：
- 按年份/月份归档
- 作为历史数据用于数据分析
- 不参与日常编辑

---

### docs/课程资料包/ - 教程文档（根目录）

**存放位置**：项目根目录`docs/课程资料包/`

**内容**：Claude Code完整课程（23个文档，253,000字）

**管理规则**：
- 独立维护，不参与公众号发布
- 用于学习和分享

---

## 快速操作

### 发布文章
```bash
# 1. 写作（自动保存到drafts/）
/write [主题]

# 2. 质量检查
/pre-check

# 3. 发布后移至archive/
mv articles/drafts/2025-12-12_xxx.md archive/2025/12/
```

### 数据分析
```bash
# 基于archive/中的已发布文章
/data-collect
/data-analyze
```
