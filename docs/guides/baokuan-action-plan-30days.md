## 🔥 P0：立即可做（Week 1-2，7天）

### Gap 1：增加互动数据指标 ⚠️ 最重要

**为什么紧急**：2025年算法看重互动数据，不是阅读量！

- [ ] **Task 1.1**：修改`data_collector`脚本
  - 增加采集：点赞数、在看数、分享数、评论数、划线数
  - 文件：`.claude/skills/gongzhonghao-writer/scripts/collectors/collect_time_range.py`
  - 工作量：2小时

- [ ] **Task 1.2**：修改`analysis_report.json`结构
  - 增加字段：`interaction_metrics`（互动指标）
  - 计算：互动率 = (点赞+在看+分享) / 阅读量
  - 工作量：1小时

- [ ] **Task 1.3**：更新`formulas_config.json`
  - 增加互动数据在爆款判断中的权重
  - 新公式：爆款 = (阅读量 × 0.4) + (互动率 × 0.6)
  - 工作量：1小时

### Gap 2：建立内容结构规范 ⚠️ 关键

**为什么紧急**：标题好只是第一步，内容质量决定互动！

- [ ] **Task 2.1**：创建"开篇3秒法则"Checklist
  - 制造悬念型：`examples/opening_suspense.md`
  - 痛点型：`examples/opening_pain.md`
  - 冲突型：`examples/opening_conflict.md`
  - 工作量：2小时

- [ ] **Task 2.2**：创建"情绪波动曲线"模板
  - 五大核心情绪：愤怒、怀旧、愧疚、暖心、焦虑
  - 混合策略：先痛点→再解决→最后升华
  - 文件：`prompts/templates/emotion_curve.md`
  - 工作量：2小时

- [ ] **Task 2.3**：创建"结尾金句生成器"
  - 收集Top 50金句案例
  - 提炼公式：`[痛点总结] + [希望/力量] + [行动号召]`
  - 文件：`scripts/core/golden_sentence_generator.py`
  - 工作量：3小时

### Gap 9：前30分钟互动预警 ⚠️ 算法关键

**为什么紧急**：前30分钟决定算法是否继续推荐！

- [ ] **Task 3.1**：建立"前30分钟监控"机制
  - 发布后30分钟内监控互动数据
  - 如果互动率<5%，发送预警
  - 工具：`scripts/monitors/first_30min_monitor.py`
  - 工作量：3小时

## ✅ P1：短期优化（Week 2-3，14天）

### Gap 3：建立选题时效性判断

- [ ] **Task 4.1**：编写"热点时效判断器"
  - 三个窗口：6小时黄金期 / 24-48小时发酵期 / 3天后长尾期
  - 每个窗口给出建议（适合类型、竞争程度）
  - 集成到`/topic-filter`命令
  - 工作量：4小时

- [ ] **Task 4.2**：编写"选题淘汰机制"（仿咪蒙50:1）
  - 评估标准：时效性、独特角度、用户需求、竞争程度
  - 自动打分并给出"建议写"或"不建议写"
  - 工作量：3小时

### Gap 10：白三角选题法

- [ ] **Task 5.1**：建立"白三角选题工具"
  - 左侧：内容方向（工具/教程/案例/资讯）
  - 右侧：用户兴趣（从互动数据分析得出）
  - 交集：爆款潜力区域
  - 工具：`scripts/core/white_triangle_selector.py`
  - 工作量：4小时

### Gap 6：标题测试机制

- [ ] **Task 6.1**：建立"标题A/B测试"工具
  - 为每篇文章准备5-10个标题备选
  - 在小范围群体（50-100人）投票
  - 选择得票最高的发布
  - 工具：`scripts/core/title_ab_test.py`
  - 工作量：4小时

### Gap 5：数据迭代反馈

- [ ] **Task 7.1**：建立"每篇文章深度分析"模板
  - 仿咪蒙1万字分析报告（简化版5000字）
  - 包括：选题分析、标题测试、内容节奏、互动数据、改进方向
  - 模板：`templates/deep_analysis_template.md`
  - 工作量：5小时

- [ ] **Task 7.2**：编写"因果关系拆解"脚本
  - 对比同类型文章
  - 拆解：为什么这篇爆了？哪个环节做对了？
  - 形成可复制规律
  - 工作量：4小时

### Gap 4：转发动机设计

  - 工作量：4小时

- [ ] **Task 9.2**：建立"品牌一致性检查器"
  - 检查每篇文章是否符合品牌调性
  - 检查是否偏离价值观

  - 工作量：3小时

### Gap 8：情绪设计系统化

  - 工作量：5小时
